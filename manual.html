충청남도의회AI의정브레인

사용자 매뉴얼

충청남도의회 의원 및 사무처 직원을 위한 AI 기반 의정활동 지원 시스템 이용 안내서

버전 v3.0 (신규 UI) 작성일 2026-06-25

1. 개요

AI의정브레인은 충청남도의회 의원 및 사무처 직원의 예산·결산 분석, 회의록 검색, 행정사무감사 질의 생성 등 의정활동을 지원하기 위해 구축된 AI 시스템입니다.

자체 구축한 거대언어모델(LLM)과 충남도청·충남교육청 관련 문서를 학습한 검색증강생성(RAG) 데이터베이스를 기반으로, 사용자가 입력한 질문에 대해 관련 자료를 검색하고 근거를 함께 제시하는 답변을 생성합니다.

⚠ 답변 활용 시 유의 AI 답변은 의사결정을 위한 참고자료입니다. 최종 의사결정 전에는 반드시 원문 자료와 소관부서를 통해 사실관계를 확인하시기 바랍니다.

2. 화면 구성

2.1 좌측 사이드바

영역설명
새 대화현재 대화를 종료하고 새로운 대화를 시작합니다. 주제가 바뀌면 새 대화로 시작하는 것을 권장합니다.
대화 검색이전에 나눈 대화 내용을 키워드로 검색합니다.
이전 대화최근 대화 목록이 최신순으로 표시됩니다. 항목을 클릭하면 해당 대화로 이동합니다.
하단 계정 영역로그인한 기관/사용자명(예: 충청남도의회) 표시, 도움말(?), 설정(⚙), 로그아웃(⏏) 아이콘이 위치합니다.

2.2 메인 대화 영역

  • 환영 메시지: 로그인한 사용자명에 맞춰 환영 문구가 표시됩니다.
  • 질문 입력창: 화면 하단 입력창에 질문을 입력합니다.
  • 자료 업로드: PDF, 한글, 엑셀 등 파일을 첨부하여 해당 자료를 기반으로 질의할 수 있습니다.
  • 전송 버튼: 입력 완료 후 클릭하거나 Enter 키로 질문을 전송합니다.

2.3 도청/교육청 모드 전환

입력창 우측 하단의 충남도청 드롭다운에서 질의 대상 기관을 전환할 수 있습니다.

충남도청 모드

충청남도 예산·결산서 및 사업설명서 등 도청 소관 지식베이스를 기준으로 답변합니다.

충남도교육청 모드

충청남도교육청 예산·결산서 및 사업설명서 등 도교육청 소관 지식베이스를 기준으로 답변합니다.

행정사무감사 모드

행정사무감사요구자료 답변서 를 기준으로 검색 및 분석, 질의서 작성을 제공합니다.

회의록 모드

제12대 충청남도의회 회의록 지식베이스를 기준으로 회기별, 의원별 회의록 요약 기능을 제공합니다.

💡 모드 선택 주의 모드를 잘못 선택하면 무관한 기관의 자료를 기반으로 답변이 생성될 수 있습니다. 질문 전 반드시 대상 기관을 확인하세요.

3. 기본 사용법

3.1 새 대화 시작

  1. 좌측 사이드바 상단의 새 대화를 클릭합니다.
  2. 메인 화면에 환영 메시지와 빈 입력창이 표시됩니다.

3.2 질문 입력 및 모드 선택

  1. 입력창 우측 하단에서 모델(충남도청 / 충남교육청 / 행정사무감사 / 회의록 등)을 선택합니다.
  2. 입력창에 질문을 작성합니다.
  3. Enter 또는 전송 아이콘(➤)을 클릭합니다.
  4. 답변 생성 중에는 로딩 표시가 나타나며, 완료되면 본문과 함께 출처보기(해당 시)가 표시되며, 해당 인용파일 선택시 오른쪽화면에서 확인이 가능합니다.

3.3 자료 업로드

  1. 입력창 좌측 하단 자료 업로드 버튼을 클릭합니다.
  2. 분석을 원하는 파일을 선택합니다.
  3. 업로드 완료 후 해당 파일에 대한 질문을 입력합니다.
⚠ 업로드 자료 보안 대외비·미공개 자료 업로드 시 소관부서 보안 지침을 우선 확인하십시오. 자료 업로드는 분석시 추가자료가 필요할 때 사용하세요.

3.4 대화 검색 / 이전 대화 불러오기

  • 대화 검색: 키워드 입력 시 일치하는 과거 대화가 목록으로 표시됩니다.
  • 이전 대화 목록: 사이드바에서 직접 클릭하여 이어서 질문할 수 있습니다.

4. 주요 기능

4.1 답변 운영 모드

모드용도출력 형태
Mode A 질문&답변 특정 사업 및 관련 예산 항목에 대한 질의응답, 검색 및 확인 근거 인용 포함 답변

모드는 질문의 형태(예: "~보고서 작성해줘" vs "~얼마야?")에 따라 시스템이 자동으로 판단하여 적용합니다.

💡 근거 확인 답변 하단에 표시되는 출처(문서명·페이지 등)를 통해 원문을 대조 확인할 수 있습니다.

4.3 업무별 활용 예시

예산·결산 분석

"2026년 본예산현황을 확인하고, 현황을 그래프로 그려줘"

사업 검색

"어린이집 관련 사업들을 찾아줘", "OOOOO 사업의 사업설명서 보여줘", "위 사업의 문제점은 없는지 분석해줘", "위 분석을 바탕으로 보고서를 작성해줘"

행정사무감사 요구자료 답변 검색 및 질의서 작성

"OOO 의원의 요구자료 목록을 확인해줘", "OOOOOOOOO 자료를 확인해줘", "위 자료를 분석해서 문제점은 없는지 확인하고 행정사무감사에서 질의할 질의서를 작성해줘"

회의록 검색

"제OOO회 임시회에서 OOO의원의 발언내역을 확인해줘", "OOO관련 발언에 대해 검색해줘"

5. 효과적인 질문 작성법

구분예시
✅ 권장"2026년 1회추경 기준 기획조정실의 세출현황을 확인해줘"
❌ 비권장"결산 좀 보여줘" (대상 연도·부서·항목 불명확)
  • 회계연도, 소관 부서(국·과), 사업명 등 구체적 범위를 명시합니다.
  • 표·요약·보고서 등 원하는 출력 형식을 함께 요청하면 결과 품질이 높아집니다.
  • 이전 답변에 이어 추가 분석이 필요하면 새 대화 대신 같은 대화창에서 후속 질문을 입력합니다.

6. 이용 시 유의사항

⚠ AI 답변의 한계 AI는 학습된 자료 범위 내에서 답변하며, 수치 오인식 등의 오류가 발생할 수 있습니다. 의사결정·대외 공표 전 반드시 원문 및 소관부서 확인을 거쳐야 합니다.
  • 대외비·개인정보가 포함된 자료는 업로드 전 소관부서 보안 지침을 확인합니다.
  • 생성된 보고서·질의서는 반드시 담당자 검토 후 사용합니다.
  • 시스템 오류·이상 응답 발견 시 문의처(예산정책담당관실: 041-635-5204)로 신고합니다.

7. 자주 묻는 질문(FAQ)

충남도청 모드와 충남교육청 모드를 동시에 질문할 수 있나요?
한 번에 하나의 모드만 적용됩니다. 두 기관 자료를 모두 비교하려면 모드를 전환하여 각각 질문 후 답변을 비교하시기 바랍니다.
업로드한 자료는 어떻게 처리되나요?
업로드 자료는 해당 대화 내 질의응답 처리에 활용됩니다. 보존·삭제 정책은 소관부서(예산정책담당관실) 안내를 따릅니다.
답변에 인용된 출처가 실제 자료와 다른 것 같습니다.
드물게 인용 오류(잘못된 출처 표기)가 발생할 수 있습니다. 반드시 원문과 대조 확인하시고, 반복되는 오류는 문의처(예산정책담당관실: 041-635-5204)로 사례를 신고해 주십시오.
이전 대화는 언제까지 보관되나요?
보관 기간은 시스템 운영 정책에 따라 결정됩니다. 장기 보관이 필요한 결과물은 별도로 다운로드·저장하시기 바랍니다.
모바일에서도 사용할 수 있나요?
AI의정브레인 톡(모바일 챗봇)이 별도 채널로 제공될 예정입니다.(별도 안내 예정)
