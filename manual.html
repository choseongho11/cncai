01. 시스템 개요

의원님의 의정활동, AI가 보좌합니다.

'충청남도의회 AI 의정브레인'은 수만 페이지의 예산서, 복잡한 자치법규, 방대한 회의록 데이터를 실시간으로 분석합니다. 단순한 검색을 넘어, 데이터 사이의 모순점을 찾고 대안을 제시하는 의원님 전용 지능형 어시스턴트입니다.

초고속 분석

방대한 데이터를 수 초 내 분석하여 핵심 요약 제공

자동 초안 작성

조례안, 질의 요지 등 문서 초안 자동 생성

데이터 검증

집행부 답변의 논리적 허점 및 수치 오류 포착

04. 예산·사업 분석 실전

AI에게 이렇게 물어보세요

의원님 질문 예시

"최근3년간 충청남도의 본예산현황을 일반회계와 특별회계로 구분해서 확인해주고 그래프로 그려줘"

의원님 질문 예시

"해양쓰레기 관련 사업을 찾아줘"

"(사업목록을 바탕으로 추가질문)해양쓰레기정화사업 지원 사업설명서를 확인하고 해당 사업을 분석해줘"

"(분석결과 답변시 추가 요청)위 답변을 분석보고서로 작성해줘"

주요 학습 데이터

  • • 최근 3개년 충청남도청 예산서 및 결산서(2026년 본예산 포함)
  • • 충청남도교육청 교육비특별회계 예산서 및 결산서(2026년 본예산 포함)
  • • 각 부서별 주요 사업설명서 자료

질문 시 좌측 하단에 관련 근거 페이지 정보(소스)가 함께 표시됩니다.

소스정보를 클릭하시면 소스 원문을 확인하실수 있습니다.

05. 실패 없는 AI 활용 팁

프롬프트(질문)의 기술

질문은 구체적일수록 좋습니다. '육하원칙'을 적용하여 부서명, 사업명, 분석 기간을 명확히 명시하세요.

#태그 활용법

질문 끝에 #표로 정리 또는 #요약해줘를 붙이면 원하는 형식으로 답변합니다.

팩트체크 가이드

AI는 드물게 수치 계산 오류를 범할 수 있습니다(할루시네이션). 최종 보고 시에는 다음을 확인하세요.

  • AI가 답변 하단에 제공한 원본 참조 링크 확인
  • 주제가 바뀔 때는 반드시 [➕ 새 채팅] 클릭
  • 복잡한 수치는 제공된 원문 PDF와 대조
